# SuperSources

Doel van het SuperSources template is om een uniforme standaard voor multi-source layouts te creÃ«ren. De gedachte is dat deze in verschillende applicaties universeel toegepast kan worden.

Features:

- Er worden geen 'halve' pixels gebruikt.
- Uniforme afstanden tussen lagen
- De specificaties zijn op de volgende manieren hier onder beschikbaar:
  - Images
  - Pixel maten tabel
  - Atem maten tabel (to do)
